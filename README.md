# 模块介绍
- shop-parent：父工程
- shop-api：公共接口
- shop-common：工具工程
- shop-pojo：实体
- shop-coupon-service：优惠券服务
- shop-goods-service：商品服务
- shop-order-service：订单服务
- shop-order-web：订单系统
- shop-pay-service：支付服务
- shop-pay-web：支付系统
- shop-user-service：用户服务
